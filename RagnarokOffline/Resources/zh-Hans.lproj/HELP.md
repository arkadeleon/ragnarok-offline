# 将 Ragnarok Online 客户端文件复制到设备

## 方法一：Wi-Fi（从 Windows 到 iPhone/iPad）

**第一步 — 在 Windows 上共享文件夹**

1. 右键点击 kRO 客户端文件夹，选择**属性**
2. 打开**共享**选项卡，点击**高级共享**
3. 勾选**共享此文件夹**，点击**确定**
4. 打开**设置** → **网络和 Internet**，确认网络配置文件设置为**专用**
5. 在**控制面板** → **网络和共享中心** → **更改高级共享设置**中，启用**网络发现**和**文件和打印机共享**

**第二步 — 查找 PC 的 IP 地址**

按 **Win + R**，输入 `cmd`，按 Enter，然后输入 `ipconfig`。记下 **IPv4 地址**（例如 `192.168.1.10`）。

**第三步 — 从 iOS 连接**

1. 打开**文件** App
2. 点击右上角的 **···**，选择**连接服务器**
3. 输入 `smb://192.168.1.10`（替换为你的 PC IP 地址）
4. 输入 Windows 用户名和密码，点击**连接**

**第四步 — 复制文件**

将以下内容复制到**我的 iPhone/iPad → Ragnarok Offline**（长按每个项目 → **拷贝**，导航到目标位置 → **粘贴**）：

- **data.grf**（来自 kRO 根目录）
- **BGM** 文件夹（来自 kRO 根目录）
- **System** 文件夹（来自 kRO 根目录）

---

## 方法二：USB 数据线

**Mac：** 连接设备 → 打开 **Finder** → 选择 iPhone/iPad → **文件**选项卡 → 展开 **Ragnarok Offline** → 拖入文件。

**Windows：** 连接设备 → 打开 **iTunes** → 选择设备 → **文件共享** → 选择 **Ragnarok Offline** → 拖入文件。
