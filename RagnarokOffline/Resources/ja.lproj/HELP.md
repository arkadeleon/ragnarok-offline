# Ragnarok Online クライアントファイルをデバイスにコピーする

## 方法 1：Wi-Fi（Windows から iPhone/iPad へ）

**手順 1 — Windows でフォルダを共有する**

1. kRO クライアントフォルダを右クリックし、**プロパティ**を選択する
2. **共有**タブを開き、**詳細な共有**をクリックする
3. **このフォルダを共有する**にチェックを入れ、**OK** をクリックする
4. **設定** → **ネットワークとインターネット**を開き、ネットワークプロファイルが**プライベート**になっていることを確認する
5. **コントロールパネル** → **ネットワークと共有センター** → **共有の詳細設定の変更**で、**ネットワーク探索**と**ファイルとプリンターの共有**を有効にする

**手順 2 — PC の IP アドレスを確認する**

**Win + R** を押して `cmd` と入力し、Enter を押した後、`ipconfig` と入力する。**IPv4 アドレス**（例：`192.168.1.10`）をメモする。

**手順 3 — iOS から接続する**

1. **ファイル**アプリを開く
2. 右上の **···** をタップして**サーバへ接続**を選択する
3. `smb://192.168.1.10`（PC の IP アドレスに置き換える）を入力する
4. Windows のユーザー名とパスワードを入力し、**接続**をタップする

**手順 4 — ファイルをコピーする**

以下のアイテムを **この iPhone/iPad 内 → Ragnarok Offline** にコピーする（各アイテムを長押し → **コピー**、移動先に移動 → **ペースト**）：

- **data.grf**（kRO のルートフォルダから）
- **BGM** フォルダ（kRO のルートフォルダから）
- **System** フォルダ（kRO のルートフォルダから）

## 方法 2：USB ケーブル

**Mac：** デバイスを接続 → **Finder** を開く → iPhone/iPad を選択 → **ファイル**タブ → **Ragnarok Offline** を展開 → ファイルをドラッグする。

**Windows：** デバイスを接続 → **iTunes** を開く → デバイスを選択 → **ファイル共有** → **Ragnarok Offline** を選択 → ファイルをドラッグする。
