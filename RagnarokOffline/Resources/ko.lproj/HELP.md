# Ragnarok Online 클라이언트 파일을 기기로 복사하기

## 방법 1: Wi-Fi (Windows에서 iPhone/iPad로)

**1단계 — Windows에서 폴더 공유하기**

1. kRO 클라이언트 폴더를 마우스 오른쪽 버튼으로 클릭하고 **속성**을 선택한다
2. **공유** 탭을 열고 **고급 공유**를 클릭한다
3. **이 폴더 공유**에 체크하고 **확인**을 클릭한다
4. **설정** → **네트워크 및 인터넷**을 열고 네트워크 프로필이 **개인**으로 설정되어 있는지 확인한다
5. **제어판** → **네트워크 및 공유 센터** → **고급 공유 설정 변경**에서 **네트워크 검색**과 **파일 및 프린터 공유**를 활성화한다

**2단계 — PC의 IP 주소 확인하기**

**Win + R**을 누르고 `cmd`를 입력한 후 Enter를 누른다. 그런 다음 `ipconfig`를 입력한다. **IPv4 주소**(예: `192.168.1.10`)를 메모한다.

**3단계 — iOS에서 연결하기**

1. **파일** 앱을 열기
2. 오른쪽 상단의 **···**를 탭하고 **서버에 연결**을 선택한다
3. `smb://192.168.1.10`을 입력한다 (PC의 IP 주소로 대체)
4. Windows 사용자 이름과 비밀번호를 입력하고 **연결**을 탭한다

**4단계 — 파일 복사하기**

다음 항목을 **내 iPhone/iPad → Ragnarok Offline**으로 복사한다 (각 항목을 길게 누르기 → **복사**, 목적지로 이동 → **붙여넣기**):

- **data.grf** (kRO 루트 폴더에서)
- **BGM** 폴더 (kRO 루트 폴더에서)
- **System** 폴더 (kRO 루트 폴더에서)

---

## 방법 2: USB 케이블

**Mac:** 기기를 연결 → **Finder** 열기 → iPhone/iPad 선택 → **파일** 탭 → **Ragnarok Offline** 펼치기 → 파일을 드래그한다.

**Windows:** 기기를 연결 → **iTunes** 열기 → 기기 선택 → **파일 공유** → **Ragnarok Offline** 선택 → 파일을 드래그한다.
