# 將 Ragnarok Online 客戶端檔案複製到裝置

## 方法一：Wi-Fi（從 Windows 到 iPhone/iPad）

**第一步 — 在 Windows 上共用資料夾**

1. 右鍵點擊 kRO 客戶端資料夾，選擇**內容**
2. 開啟**共用**索引標籤，點擊**進階共用**
3. 勾選**共用此資料夾**，點擊**確定**
4. 開啟**設定** → **網路和網際網路**，確認網路設定檔為**私人**
5. 在**控制台** → **網路和共用中心** → **變更進階共用設定**中，啟用**網路探索**和**檔案及印表機共用**

**第二步 — 查找 PC 的 IP 位址**

按 **Win + R**，輸入 `cmd`，按 Enter，然後輸入 `ipconfig`。記下 **IPv4 位址**（例如 `192.168.1.10`）。

**第三步 — 從 iOS 連線**

1. 開啟**檔案** App
2. 點擊右上角的 **···**，選擇**連線伺服器**
3. 輸入 `smb://192.168.1.10`（替換為你的 PC IP 位址）
4. 輸入 Windows 使用者名稱和密碼，點擊**連線**

**第四步 — 複製檔案**

將以下內容複製到**我的 iPhone/iPad → Ragnarok Offline**（長按每個項目 → **拷貝**，導航到目標位置 → **貼上**）：

- **data.grf**（來自 kRO 根目錄）
- **BGM** 資料夾（來自 kRO 根目錄）
- **System** 資料夾（來自 kRO 根目錄）

---

## 方法二：USB 傳輸線

**Mac：** 連接裝置 → 開啟 **Finder** → 選擇 iPhone/iPad → **檔案**索引標籤 → 展開 **Ragnarok Offline** → 拖入檔案。

**Windows：** 連接裝置 → 開啟 **iTunes** → 選擇裝置 → **檔案共享** → 選擇 **Ragnarok Offline** → 拖入檔案。
