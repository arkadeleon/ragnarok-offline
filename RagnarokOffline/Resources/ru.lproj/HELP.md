# Копирование файлов клиента Ragnarok Online на устройство

## Способ 1: Wi-Fi (с Windows на iPhone/iPad)

**Шаг 1 — Открыть общий доступ к папке на Windows**

1. Щёлкните правой кнопкой мыши по папке клиента kRO и выберите **Свойства**
2. Откройте вкладку **Доступ** и нажмите **Расширенная настройка**
3. Установите флажок **Открыть общий доступ к папке** и нажмите **ОК**
4. Откройте **Параметры** → **Сеть и Интернет** и убедитесь, что сетевой профиль установлен как **Частный**
5. В **Панели управления** → **Центр управления сетями и общим доступом** → **Изменить дополнительные параметры общего доступа** включите **Сетевое обнаружение** и **Общий доступ к файлам и принтерам**

**Шаг 2 — Узнать IP-адрес PC**

Нажмите **Win + R**, введите `cmd`, нажмите Enter, затем введите `ipconfig`. Запишите **IPv4-адрес** (например, `192.168.1.10`).

**Шаг 3 — Подключиться с iOS**

1. Откройте приложение **Файлы**
2. Нажмите **···** в правом верхнем углу и выберите **Подключиться к серверу**
3. Введите `smb://192.168.1.10` (используйте IP-адрес вашего PC)
4. Введите имя пользователя и пароль Windows, затем нажмите **Подключиться**

**Шаг 4 — Скопировать файлы**

Скопируйте следующие элементы в **На моём iPhone/iPad → Ragnarok Offline** (нажмите и удерживайте каждый элемент → **Скопировать**, перейдите в папку назначения → **Вставить**):

- **data.grf** (из корневой папки kRO)
- Папка **BGM** (из корневой папки kRO)
- Папка **System** (из корневой папки kRO)

## Способ 2: USB-кабель

**Mac:** Подключите устройство → откройте **Finder** → выберите iPhone/iPad → вкладка **Файлы** → разверните **Ragnarok Offline** → перетащите файлы.

**Windows:** Подключите устройство → откройте **iTunes** → выберите устройство → **Общий доступ к файлам** → выберите **Ragnarok Offline** → перетащите файлы.
